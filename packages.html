Coq Package Index

Index of Coq Packages

Coq developments (formalizations, plugins) can be packaged using OPAM. OPAM makes it easy to install a package by taking care of the compilation process and resolving all package dependencies. The Coq development team maintains an index of such packages that can be browsed online.

We provide instructions on how to install Coq and related packages via OPAM step by step.

You are encouraged to make your Coq developments visible and easily installable by submitting them to the Coq Package Index.
